Responsive Navbar Test

Test Instructions:

  1. Resize your browser window to test different screen sizes
  2. On mobile sizes (≤768px), the hamburger menu should appear
  3. Click the hamburger menu to open/close the mobile navigation
  4. Test dropdown menus on both desktop (hover) and mobile (click)
  5. Check the console for debug information

Desktop (1025px+)

All navigation links should be visible horizontally. Dropdowns work on hover.

Tablet (769px - 1024px)

Navigation should still be horizontal but with adjusted spacing.

Mobile (≤768px)

Hamburger menu should appear. Navigation becomes vertical when opened.

Debug Information

Open the browser console (F12) to see detailed debug information about the responsive behavior.
